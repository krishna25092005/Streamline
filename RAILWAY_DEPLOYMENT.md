## 📦 Deployment Steps

### Method A: Deploy via GitHub (Recommended)

1. **Push Codebase to GitHub**:
   ```bash
   git add .
   git commit -m "Configure Railway production deployment"
   git push origin main
   ```

2. **Connect to Railway**:
   - Go to [railway.app/new](https://railway.app/new).
   - Select **Deploy from GitHub repo**.
   - Select your repository (`Streamline`).

3. **Add Environment Variables**:
   - Click on your deployed service in Railway.
   - Open the **Variables** tab and paste the environment variables listed above.

4. **Generate Public Domain**:
   - Go to **Settings -> Networking -> Generate Domain**.
   - Your application will be live at `https://streamline-production-xxx.up.railway.app`!

---

### Method B: Deploy via Railway CLI

1. **Install Railway CLI**:
   ```bash
   npm i -g @railway/cli
   ```

2. **Login & Deploy**:
   ```bash
   railway login
   railway init
   railway up
   ```

3. **Expose Domain**:
   ```bash
   railway domain
   ```
